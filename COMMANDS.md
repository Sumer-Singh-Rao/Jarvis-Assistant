# JARVIS Voice Commands

## Typing & Dictation
**Type Text:**
- "write Hello World" - Types the text
- "type This is a test message" - Types text
- "write Dear Sir, I am writing to inform you..." - Long text

**Keyboard Shortcuts:**
- "press enter" or "enter" - Press Enter key
- "backspace" - Delete one character
- "backspace 5" - Delete 5 characters
- "select all" - Select all text (Ctrl+A)
- "copy" - Copy selected text (Ctrl+C)
- "paste" - Paste text (Ctrl+V)
- "cut" - Cut selected text (Ctrl+X)
- "undo" - Undo last action (Ctrl+Z)
- "redo" - Redo last action (Ctrl+Y)

**Usage Example:**
1. Open notepad: "open notepad"
2. Type text: "write Hello, this is JARVIS typing for you"
3. Press enter: "press enter"
4. Type more: "write This is amazing!"
